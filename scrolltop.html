The jQuery scrollTop method is used to set or return the vertical scrollbar position for the selected element. When the scrollbar is on the top, it specifies the position 0. To return the position: When this method is used to return the position, it provides the current vertical position of the first matched element in the set. To set the position: When this method is used to set the position, it sets the vertical position of the scrollbar for all matched element.
